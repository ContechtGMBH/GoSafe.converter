### GoSafe Rail Converter

Rest api for GoSafe Rail. Allows to convert popular formats to subgraph that can be a part of the main graph representing the network topology.

#####Modules:
- RailML converter
- IFCRail - TODO
- Others - TODO

#####Requirements:
- Python 2.7
- Libs: flask, lxml, py2neo, pyproj
